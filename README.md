# Vektor

Progressively enhanced Pong-style arcade games in plain HTML, CSS, and JavaScript. No build step.

- **Hub** — `/` (`index.html`)
- **V1** — baseline Pong — `/v1/`
- **V2** — match play (best of 5 games, first to 7 per game), seven color themes, Web Audio sounds — `/v2/`

Deploy on Vercel with framework **Other**, no build command, output directory **root**.

Palette and structure follow the project brief; audio is synthesized with the Web Audio API (no audio files).
